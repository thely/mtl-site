Common Questions
Where should I save my files?
If you're saving your files on one of the lab stations, save it in the Data drive. You can find the Data drive by opening Finder (file viewer), and the Data drive will be available on the sidebar.
I can't hear anything.
Check where your headphones are plugged in, and move the volume knob. If that doesn't work, try switching your headphones to the other headphone port.
How do I get a program out of fullscreen mode?
Hit Control + Command + F. This will toggle between fullscreen and windowed mode.

Ableton Live 9

Troubleshooting

I can't hear anything.
Go to Ableton Live > Preferences, and click on the Audio tab. Make sure the Output and Input Devices are both set to the Scarlett 18i20. Make sure the volume knob on your headphones is up.
Why isn't the red automation line appearing to use automation in Clip View?
You probably imported the project from elsewhere, so you need to Consolidate the clip first. Either right-click on the clip and find Consolidate in the dropdown, or click on the clip and press ⌘ + J.

Descriptions

  • Arrangement View: One of two ways of adding/editing/arranging sounds in Live. Time moves from left to right.
  • Clip View: Edit/alter a selected clip. The two tabs under this region let you switch between Clip View (displayed) or Device View (add/edit audio effects).
  • Transport Bar: Play/pause, stop, record, and check what measure the playhead is on.
  • File Browser: View the files in your project, and add new instruments and audio effects.
  • Info View: Hover over anything in Live, and Info View will tell you more about it. You can press ? (Shift + /) to show/hide it.

Logic Pro X

Most Logic problems can be solved with a Google search. If you're really stuck, you can always look at the full PDF manual.

Map

The screenshot below has some core areas of Logic, labeled. If you ever want to know what a button does or what something is called in Logic (there's a lot more than what this screenshot lists!), you can always check Quick Help by clicking on the question mark in the upper left (blue border in this screenshot).

Descriptions

  • Arrangement View: The core window/pane, where all regions are added/edited/arranged. Time moves from left to right.
  • Inspector: Shows the channel strip, output channel, and other details about the currently selected track and region(s).
  • Transport Bar: Buttons for moving around in the arrangement, time information, cycling, recording, etc.
  • Quick Help: Question-mark button in the upper left. Shows help information for any button or area in Logic that you hover over with your mouse.
  • File Browser: Camera/music note button in the upper right. Shows all files currently in use in the session, including what sub-selections of files, and lets you browse through files on your computer.

Finale 2014

View the Finale for Mac user manual here.

Troubleshooting

I can't hear my keyboard.
Go to MIDI/Audio > Device Setup > MIDI/Internal Speaker Setup. Make sure that the first entry under Input Device says "Axiom Pro 64" or something similar. Usually this is enough to get the keyboard to start working again.
How do I enter harmonic analysis again?
Make sure you're using the Lyrics Tool with Finale Numerics as your font. All the shortcut keys are listed on the Harmonic analysis page in the Finale manual.

Speedy Entry Reference

Using Speedy Entry in Finale makes score entry much easier, even though at first it seems difficult or annoying. Some of these key commands are similar in other notation programs like Sibelius and MuseScore, so it's worthwhile to learn them now. For all the speedy entry key commands, check the Speedy Entry page of the manual.

Entering Notes

Press the note you want the pitch of on the piano keyboard, and then press the number corresponding to the note's duration. To enter a rest, just press the number.

Practica Musica
How can I check what excercises I've done?
On the top bar, find the tab that has your name. Select [Your Name] > Progress report, and click the blue View Report button.
How do I save my progress?
It autosaves, but if you want to manually save, go to [Your Name] > Save progress now.

Logging in

  1. Click the Practica Musica icon in the Dock
  2. Select the third option: "I have a file online at WebStudents - let me choose it"
  3. Enter urmus in the WebStudents Account field, and make sure Use WebStudents is checked
  4. Pick your file from the list; filenames are formatted as LastnameFirstname360417.stu
  5. Enter your password, if you made one during account creation

EarMaster Cloud
How can I check what excercises I've done?
Go to Statistics > Results, or press Command + R.
How do I save my progress?
Your progress syncs when you quit the program. Quit EarMaster by closing it, or pressing Command + Q. A message will pop up saying "Syncing with EarMaster Cloud" - don't press Skip!

Logging in

  1. Click the EarMaster icon in the Dock
  2. Enter urmtl for school account, and your username and password.
Note: Your username and password were emailed to you at the beginning of the semester. If you lost this information, please contact the current lab manager.

General Mac Tips

Shortcuts/Key Commands

Below are some helpful shortcut keys. If you're used to using shortcut keys on a Windows computer, most of your Windows shortcuts are the same or similar, but use Command (⌘) instead of Control.

If you're looking for more, Apple has a much longer list of general key commands.

Legend

Some programs will use symbols for the modifier keys (Command, Control, etc). These are the less-obvious but frequently-seen symbols.

Symbol Key Symbol Key
⌘ Command ⇧ Shift
⌥ Option/Alt ⌃ Control
⌫ Delete ↩ Return (Enter)

General

Key Command Description
⌘ + N Create a new file of the current program.

Example: In Word, this creates a new, blank document; in Chrome, this opens a new browser window.
⌘ + O Open an existing file of the current program
⌘ + S Save the current document/file
⌘ + W Close the front window of the current program
⌘ + Q Quit the current program and close all its open windows
⌘ + Tab Switch between programs
⌘ + Spacebar Search for anything (applications, files, documents)
⌃ + Click Right click.

Note: You can also click on the upper-right hand portion of the mouse, but this will work regardless of how a mouse is set up.
⌘ + Z Undo
⌘ + ⇧ + Z or ⌘ + Y Redo most recent Undo. Check the program to see which Redo shortcut it uses.
